# Interactive Dynamic Quiz Application

A sleek, responsive, single-page web application that hosts a dynamic multi-step quiz. Built entirely with Vanilla JavaScript, semantic HTML5, and modern CSS3, this project demonstrates clean frontend state management, dynamic DOM manipulation, and asynchronous programming architectures.

🚀 **[Live Demo Link](https://sinha-amit22.github.io/interactive_quiz_app/)** *(Update this link with your actual GitHub Pages URL once live)*

---

## 🌟 Key Features

* **Dynamic DOM Architecture:** Instead of hardcoding content, quiz questions and structural layouts are fetched asynchronously and injected dynamically into the DOM.
* **Granular State Tracking:** Manages real-time data flow including question indexes, cumulative scoring matrices, user tracking, and interaction locks to prevent multiple selections.
* **Fully Responsive Design:** Uses mobile-first layouts driven by CSS Flexbox and media query breakpoints to guarantee seamless rendering across desktops, tablets, and mobile screens.
* **Fluid UI Transitions:** Features a CSS-animated linear progress tracking indicator and structural fade-in keyframes to optimize the user experience.

---

## 🛠️ Tech Stack & Concepts Demonstrated

* **Structure:** Semantic HTML5 for accessibility and clean document object architecture.
* **Styling:** CSS3 variables (custom properties) for centralized theme management, Flexbox for flexible structures, and media queries for responsive breakpoints.
* **Logic:** Vanilla JavaScript (ES6+), encompassing asynchronous `Promises` to simulate external API micro-tasks, explicit event listener scoping, and array iteration methods.
